# Job Market Intelligence & Automation Platform

## Overview

A Python-based job market intelligence platform that automates job collection, storage, analytics, and reporting across multiple job sources.

## Features

* Selenium-based job scraping
* PostgreSQL database integration
* Flask REST API services
* Duplicate detection
* Job search and filtering
* Analytics endpoints
* Scheduled automation with APScheduler
* Data migration from SQLite to PostgreSQL

## Technologies

* Python
* Flask
* PostgreSQL
* Selenium
* APScheduler
* REST APIs
* SQL

## API Endpoints

GET /jobs

GET /jobs/source/<source>

GET /analytics

GET /analytics/summary
